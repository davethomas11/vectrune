Declarative APIs, Fast Prototyping

Vectrune is a language and runtime for building REST and GraphQL APIs with memory/state, schema validation, and builtins—all in a single readable document.

@App
name = Memory Example API
version = 1.0
type = REST
run:
    skaters = load-rune skateboarders.rune
    set-memory skaters

@Schema/Skater
fields:
    name: String
    age: Integer
    style: String

@Route/GET /Skaters
run:
    skaters = get-memory skaters
    return skaters

@Route/PUT /Skaters
run:
    skaters = get-memory skaters
    new-skater = parse-json body
    validate new-skater #Skater
    append skaters.Skateboarder.record new-skater
    set-memory skaters
    return skaters

How it Works

Vectrune compiles your .rune document into a fast, testable API server with memory/state and validation built in.

Getting Started

  1. Install: brew install vectrune
  2. Run: vectrune myapi.rune
  3. Deploy: vectrune lambda package
